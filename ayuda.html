Preguntas Frecuentes



¿Como descargar el software?


Para poder descargarlo tendrá que dirigirse en apartado de softwares y oprimir el botón "comprar" el cual le mostrará la sección de pago para realizar la compra del mismo




¿Que metodos de pago aceptamos?


Aceptamos todo tipos de método de pago, tarjetas de crédito, tarjeta de debito, tarjetas prepagas, billeteras electrónicas y transferencia bancaria




¿En cuanto tiempo tendré disponible el software que adquirí?


Automáticamente después de completar con la compra se empezará a descargar el archivo ejecutable .exe del software
